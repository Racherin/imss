#Run these commands before running the code in your IDE terminal

For Windows:
1. pip3 install -r requirements.txt
2. set FLASK_ENV=development
3. set FLASK_APP=app
4. flask run

For Linux/MacOS:
1. pip3 install -r requirements.txt
2. export FLASK_ENV=development (LINUX, MACOS)
3. export FLASK_APP=app (LINUX, MACOS)
4. flask run
